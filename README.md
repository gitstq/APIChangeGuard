# APIChangeGuard

🛡️ **APIChangeGuard** - 轻量级API变更智能检测与影响分析引擎

一个零依赖的Python CLI工具，用于智能检测API变更、分析影响范围、生成兼容性报告。

## 核心功能

- 🔍 **智能Diff检测** - 自动对比API版本差异
- 📊 **影响分析** - 识别破坏性变更和潜在风险
- 📝 **变更分类** - 自动分类新增/修改/删除/废弃
- 🎯 **兼容性评分** - 量化API兼容性健康度
- 📈 **趋势追踪** - 历史变更趋势可视化
- 🔗 **CI/CD集成** - 支持GitHub Actions等流水线

## 技术栈

- Python 3.8+
- 零外部依赖（纯标准库实现）
- 支持OpenAPI/Swagger、GraphQL、REST API
